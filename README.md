# OPERATON Community Hub Release Parent

# Usage

In your pom.xml, add a parent:
```
<parent>
    <groupId>org.operaton.community</groupId>
    <artifactId>community-hub-release-parent</artifactId>
    <version>1.0.0/version>    
    <relativePath />
</parent>  
```
